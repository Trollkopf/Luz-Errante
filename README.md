# 🌒 Luz Errante

## Descripción del juego / Game Description

**ES:**  
Luz Errante es un juego de plataformas de acción en 2D desarrollado con SpriteKit, protagonizado por un caballero que debe avanzar en medio de la oscuridad recogiendo orbes de energía y esquivando enemigos y trampas. Usa gestos para moverte, rodar, atacar y sobrevivir en un entorno hostil donde la oscuridad avanza constantemente.

**EN:**  
Luz Errante is a 2D action platformer built with SpriteKit, featuring a lone knight navigating through darkness. Collect energy orbs, dodge enemies and traps, and use gestures to roll, attack, and survive as the creeping darkness threatens to engulf everything.

---

## Características principales / Main Features

- 🎮 Control por gestos: desliza para moverte, rodar o atacar.
- ⚔️ Animaciones personalizadas para cada acción del caballero.
- 💀 Enemigos y trampas dinámicas con colisiones físicas realistas.
- 🌌 Efecto parallax en el fondo para mayor profundidad visual.
- 🌑 Oscuridad progresiva que aumenta la tensión del juego.

---

## Estado actual / Current Status

El juego está en una fase jugable con todas las mecánicas básicas implementadas. Ya es posible:

- Controlar al personaje principal con gestos.
- Recoger orbes de energía.
- Evitar o recibir daño de esqueletos y trampas.
- Morir y reiniciar la partida.

## Instalación / Installation

1. Clona el repositorio:
   ```bash
   git clone https://github.com/trollkopf/luz-errante.git
   ```
2. Abre el proyecto en Xcode.
3. Conecta tu dispositivo y ejecuta el proyecto.
